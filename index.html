Smart Voyage

Angular 7 | Bootstrap

A prototype for a travel application that helps to compute the path which has the least cost, such that users can save more on their flight tickets. The application asks the user to feed the names of each city that they have to travel and the number of days they are planning to spend in each city. The application calculates the route that offers them the minimum cost of flight tickets between those cities in the interval of time mentioned by the user.
